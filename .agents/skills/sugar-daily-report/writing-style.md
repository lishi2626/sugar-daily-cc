# 白糖日报写作风格参考

## 核心原则

1. **先结论后解释** — 首句直接点明当天最重要判断。
2. **固定顺序不混写** — 国际→巴西→印度→泰国→国际结论→国内(25/26→26/27面积→广西天气→进口→政策)→国内结论。
3. **多维度覆盖** — 每国必须覆盖生产、种植面积和降雨，无数据时明确说明。
4. **专业克制** — 用数据说话，避免情绪化。
5. **避免绝对词** — 不使用"必然""一定""毫无疑问"等。
6. **区分预测和实际** — 26/27榨季预期不得写成25/26现实供给。

## 基本面固定结构

```
国际供需总判断 → 巴西(双周生产+面积+降雨) → 印度(生产+面积+季风+出口) → 泰国(生产+面积+降雨) → 美糖判断
                                                                                              ↓
国内25/26现实供给 → 26/27种植面积 → 广西天气 → 进口 → 政策 → 国内价格判断
```

## 研究员确认固定事实（不可由模型覆盖）

- 市场表现涨跌幅由Python生成，统一写"日涨幅/日跌幅/日涨跌幅"，不得缺少"日"。
- 26/27榨季甘蔗种植面积确定增加；当天未抓到新的面积文章不代表该判断失效。
- 广西当前天气暂未对甘蔗生长和产量预期形成实质影响；不得把局部降雨或低洼地块积水扩大为全区减产风险。
- 基本面不得出现"SR2609判断""郑糖判断""内盘判断"等标签式文字。
- 基本面最后结论必须分别写国际方面和国内方面，不能混成一句。

## 国际基本面总结句

当国际基本面段落前文已经使用"国际方面"作为段落开头时，结尾总结句不得再次以"国际方面"开头。

错误示例：

国际方面，全球供需方面，……
……
国际方面，全球糖市预计小幅过剩……

正确示例：

国际方面，全球供需方面，……
……
目前，全球糖市预计小幅过剩……

可使用的自然衔接词：

- 目前
- 整体来看
- 综合来看
- 当前来看

本项目当前优先使用：

"目前，全球糖市预计……"

同一自然段内不得重复使用两次"国际方面"或两次"国内方面"作为句首。

## 巴西UNICA数据口径（不可违反）

- 只允许使用UNICA Table 2中的Sugar和Share %
- Table 2是**双周数据**，必须按报告真实数据期写成双周期间
- 正确写法: "巴西中南部在最新双周期间产糖180万吨，同比增加109.48%，制糖比为40.34%"
- 禁止写法: "截至5月1日累计产糖180万吨"、"榨季累计产糖"、"累计产糖XXX万吨"
- 如果CSV中有period_start和period_end字段，必须使用该期间
- 制糖比40.34%、乙醇占比59.66%时，不得写"高制糖比"

## 印度出口政策方向规则（最高优先级）

| 事实情况 | 影响方向 | 推荐写法 |
|---------|---------|---------|
| 无新增出口信号 | 中性偏多 | "出口政策暂无新增信号，国际市场没有新增印度糖供应压力，对糖价影响中性偏多" |
| 明确新增出口 | 偏空 | "印度新增出口配额，国际市场供应增加，对国际糖价形成压制" |
| 限制出口 | 偏多 | "印度出口受限，国际市场可获得供应减少，对国际糖价形成支撑" |

### 印度出口政策禁忌

| ❌ 禁止 | 原因 |
|---------|------|
| 无新增出口时写"出口预期形成压制" | 方向错误 |
| 无新增出口时写"出口政策影响中性" | 低估偏多含义 |
| 无新增出口时写"提供明显支撑"或"主要利多" | 过度解读，只是没有新增压力 |
| 新增出口时写"形成支撑" | 方向相反 |
| 限制出口时写"形成压制" | 方向相反 |

## 泰国结论权限（无同比数据时不可越权）

当前有效数据: 截至日期、累计入榨甘蔗量、累计产糖量、出糖率

### 无同比数据时禁止写

| ❌ 禁止 | 原因 |
|---------|------|
| 同比增加 | 无上年同期对比数据 |
| 丰产超预期 | 无官方预估对比 |
| 产量处于高位 | 无历史对比基准 |
| 整体供应充裕 | 无同比或出口数据支撑 |
| 出口能力增强 | 无出口数据 |
| 出口压力增加 | 无出口数据 |
| 市场已经消化 | 无市场数据支撑 |

### 无同比数据时允许写

- "当前榨季供应规模得到官方生产数据确认"
- impact_direction: neutral_to_bearish
- confidence: medium_low

## 进口利润和现货基差方向（固定逻辑）

| 事实 | 方向 | 推荐写法 |
|------|------|---------|
| 现货升水期货 | 现货相对坚挺 | "对盘面下方形成一定支撑" |
| 配额外进口利润为正 | 进口窗口打开 | "对后续国内供应形成潜在压力" |

**禁止**: 把进口利润为正写成支撑

推荐输出: "广西白糖现货升水郑糖主力合约XX元/吨，对盘面下方形成一定支撑；配额外进口利润为正，进口窗口打开，对后续供应形成潜在压力。"

## 全球供需写作规则（只写2026/27年度）

### 核心原则
- 日报只围绕**2026/27年度**全球供需预估
- 主观点 = 该年度中**发布日期最新**的一条机构预估
- 不按机构固定优先级（ISO不一定为主观点）

### 数据来源表述

| 数据情况 | 正文写法 |
|---------|---------|
| 有过剩数值 | "全球供需方面，Czarnikow最新预计2026/27年度全球糖市将出现约140万吨的小幅过剩" |
| 有短缺数值 | "全球供需方面，ISO最新预计2026/27年度全球糖市短缺约26万吨" |
| 只有方向 | "全球供需方面，StoneX最新预计2026/27年度全球糖市将从过剩转向短缺" |
| 平衡 | "全球供需方面，Copersucar预计2026/27年度全球糖市供需基本平衡" |
| 补充观点 | "不过StoneX认为2026/27年度将转为短缺约55万吨，机构观点存在分歧" |
| 无目标年度数据 | "全球供需方面，当前尚未获得可验证的2026/27年度最新机构预估，暂不作新增判断" |

### 全球供需禁忌

| ❌ 禁止 | 原因 |
|---------|------|
| "ISO预计2026/27年度过剩224.4万吨" | 224.4万吨是2025/26年度数据 |
| "ISO最新预计全球糖市过剩224.4万吨" | 如果上下文在分析2026/27年度 |
| "全球供需暂无新增数据" | 如果CSV中有2026/27年度观点 |
| 混写不同年度观点 | 日报只围绕2026/27年度 |
| 使用旧文章覆盖新文章 | 选择依据=目标年度+发布日期最新 |

## 美糖判断逻辑

美糖总结必须与三国方向一致：

| 三国方向 | 美糖判断 |
|---------|---------|
| 巴西=偏空, 印度=中性偏多, 泰国=neutral_to_bearish | 巴西供应压力较为明确，印度暂无新增出口未形成新的供应压制，美糖短期维持低位承压 |
| 巴西=偏空, 印度=偏空, 泰国=偏空 | 三国均形成供应压力，美糖承压明显 |
| 巴西=偏空, 印度=偏多, 泰国=偏空 | 巴西泰国压力 vs 印度支撑，美糖震荡 |

## 泰国写作规则

### 数据来源表述

| 数据情况 | 正文写法 |
|---------|---------|
| SugarZone累计产量 | "截至YYYY年M月D日，2025/26榨季全国累计入榨甘蔗约XXX万吨，累计产糖约XXX万吨" |
| 无同比数据 | "当前榨季供应规模得到官方生产数据确认" |
| 无数据 | "官方尚未公布2025/26榨季可验证的全国产量数据" |

## 国内写作规则

### 数据来源表述（优先级: 全国→广西→云南）

| 数据情况 | 正文写法 |
|---------|---------|
| 有实际产糖量/销量/库存数据 | "截至最新数据期，广西累计产糖……万吨，累计销糖……万吨，工业库存……万吨，产销率……%" |
| 有预估数据 | "沐甜预计广西25/26榨季最终产糖约770万吨，目前已进入纯销售期" |
| 有缓存数据(valid_cached) | "国内25/26榨季沿用最近一期已确认或预估数据。广西方面，……" |
| 有广西数据但无全国数据 | 直接使用广西数据，不需要"全国数据缺失"的判空 |
| 无任何有效数据 | "国内25/26榨季暂无新的已验证产销和库存数据，暂不对现实供应变化作新增判断" |
| 26/27种植面积 | "26/27榨季甘蔗种植面积确定增加" |
| 广西天气 | "广西当前天气暂未对甘蔗生长和产量预期形成实质影响，后续继续关注持续强降雨及局部积水风险" |
| 仅有局部天气 | "广西局部存在积水风险，后续仍需跟踪，但当前尚未形成实质影响" |

### 预估 vs 实际

| 数据类型 | 必须保留的表述 | 禁止写法 |
|---------|-------------|---------|
| forecast | 预计、预估、有望、大概率 | "最终产糖770万吨已确认" |
| actual | 累计产糖、累计销糖、工业库存 | 把预估写成实际 |

### 国内写作禁忌

| ❌ 禁止 | 原因 |
|---------|------|
| 无产销数据写"榨季已结束""格局已定" | 无数据不得推断 |
| 把预估写成实际 | data_type=forecast必须保留"预计" |
| 当天无新文章就写"暂无数据" | 应使用最近有效缓存 |
| 26/27榨季种植面积写"暂无数据" | 研究员已确认面积增加 |
| 广西天气写成全区减产压力 | 研究员确认当前尚未形成实质影响 |
| 用座谈会替代产量数据 | 会议≠生产数据 |
| 局部天气写"影响范围有限""不会影响全区" | 无全区监测数据 |
| 无面积数据写"面积保持稳定" | 无数据不得虚构 |
| 有广西数据时写"全国数据缺失" | 直接使用广西数据 |
| 出现"SR2609判断""郑糖判断""内盘判断" | 标签式表达，需改为自然段 |

## 风险提示规则

- 风险提示固定读取配置或研究员模板: "宏观、政策、天气、进口量。"
- 除非CSV中存在明确且有效的风险事件，否则不得自动生成具体事件
- 禁止自行扩写: "印度若突击批准出口""6月下旬进口糖到港""巴西雷亚尔波动""台风季影响"

## Markdown格式规则

- 基本面正文开头写"国际方面，"，不要出现"**基本面：****国际方面**"这样的连续加粗标记
- 国内部分直接接续为"国内方面，"，不要再次使用加粗小标题
- 保持日报样例的连续文字形式

## 表达对照

| ❌ 避免 | ✅ 推荐 |
|---------|--------|
| 必然上涨/下跌 | 上行动能较强/面临回调压力 |
| 毫无疑问 | 从当前数据来看 |
| 百分之百确定 | 大概率维持……格局 |
| 将预测写成已实现 | 机构预估……/市场预期…… |
| 国际国内混写 | 国际部分:……国内部分:…… |
| 美糖判断用中国数据 | 美糖判断只用国际数据 |
| 印度无新增出口写成压制 | 印度无新增出口写中性偏多 |
| 巴西双周数据写成累计 | 巴西在最新双周期间产糖…… |
| 泰国无同比写"同比增加" | 泰国当前榨季供应规模得到确认 |
| 进口利润为正写成支撑 | 进口利润为正，对后续供应形成潜在压力 |

## 数字使用规则

- 所有数据保留原始单位
- 不得补充输入之外的新数字
- 配额外进口利润保留来源、日期和参考现货市场
- 不得将日照现货利润与南宁基差混用
