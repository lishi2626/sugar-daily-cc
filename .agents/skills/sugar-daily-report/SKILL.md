## 基本面固定分析顺序

国际部分:
```
国际供需总判断 → 巴西(生产+面积+降雨) → 印度(生产+面积+季风+出口) → 泰国(生产+面积+降雨) → 美糖判断
```

国内部分:
```
25/26榨季现实供给 → 26/27种植面积 → 26/27降雨墒情苗情 → 进口/现货/基差/政策 → SR2609判断
```

每国必须覆盖生产、种植面积和降雨，无数据时明确说明。

## 印度出口政策方向规则（最高优先级）

| 事实情况 | 影响方向 | 推荐写法 |
|---------|---------|---------|
| 无新增出口信号 | 中性偏多 | "印度出口政策暂无新增信号，国际市场暂未面临新的印度糖供应增量，对糖价影响中性偏多" |
| 明确新增出口 | 偏空 | "印度新增出口配额，国际市场供应增加，对国际糖价形成压制" |
| 限制出口 | 偏多 | "印度出口受限，国际市场可获得供应减少，对国际糖价形成支撑" |

**禁止**: 无新增出口时写"出口预期形成压制"或"出口政策影响中性"

## 巴西分析覆盖要求

每期必须覆盖:
1. 当前压榨和食糖产量
2. 制糖比和糖醇价差
3. 新榨季甘蔗种植或可收获面积
4. 主产区降雨和收割天气

## 印度分析覆盖要求

每期必须覆盖:
1. 累计产糖量和同比
2. 甘蔗种植面积和主产邦进度
3. 西南季风到达时间和降雨距平
4. 出口政策

## 泰国分析覆盖要求

每期必须覆盖:
1. 累计产糖量
2. 甘蔗种植面积
3. 主产区降雨和干旱
4. 出口情况

## 泰国数据源优先级

1. **SugarZone生产报告** — 全国分区生产报告（sugarzone.in.th）
2. **OCSB Open Data** — 结构化数据（opendata.ocsb.go.th）
3. **泰国政府PRD** — 工业部/OCSB官方新闻稿（prd.go.th）
4. **OCSB主站** — 官方公告（ocsb.go.th）
5. **OCSB官方Facebook** — 公开内容（仅辅助）
6. **OCSB区域推广中心** — 地区信息（仅区域，不扩展为全国）
7. **CSV未过期OCSB缓存** — 未过期官方数据
8. **官方预测** — 最后补充（必须标注forecast）

## SugarZone解析规则

- 页面URL: https://sugarzone.in.th/production-report-by-region/
- 泰国佛历转换: 公历年 = 佛历年 - 543
- 选择最新一期报告的PDF
- PDF全国总计行标记: "รวมทั้งสิ้น"（需Unicode标准化+模糊匹配）
- 甘蔗总量单位: 吨（直接读取）
- 食糖总量单位: 100公斤/袋（需×100÷1000÷10000换算为万吨）

## coverage_scope使用规则

| scope | 可用于 | 写法示例 |
|-------|--------|---------|
| national | 整体供需判断 | "全国甘蔗种植面积……" |
| province | 省级判断 | "广西甘蔗种植面积……" |
| major_production_area | 主产区判断 | "主产区降雨……" |
| local | 仅局部影响 | "部分蔗区……" |

## 国内文章筛选

### 必须排除的文章类型
座谈会、论坛、研讨会、大会、产业升级、签约、招商、会议、领导调研、企业活动、品牌活动

### 第一优先关键词（现实生产和产销数据）
累计产糖、产混合糖、累计销糖、工业库存、产销率、入榨甘蔗、收榨、开榨

### 第二优先关键词（26/27种植和天气）
种植面积、甘蔗面积、新植蔗、宿根蔗、苗情、出苗、分蘖、墒情、高温、强降雨、干旱

## 关键规则

- SR0 ≠ SR2609，不得改名
- 市场表现写"郑糖主力合约"
- 印度出口政策方向: 无新增出口=中性偏多, 新增出口=偏空, 限制出口=偏多
- 美糖判断必须与三国方向一致
- 泰国区域数据只能写"泰国部分产区……"
- 国内不得用会议资讯替代产量、销量和库存数据
- 局部天气只能写"部分蔗区"，不得扩大为全区判断
- 无数据时写"暂无新的已验证信息"，不得虚构
